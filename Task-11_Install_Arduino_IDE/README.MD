# Task 11 - Install Arduino IDE

## Aim
To install the Arduino IDE software required for programming Arduino boards.

## Software Required
- Arduino IDE
- Internet Connection
- Windows/Linux/Mac Computer

## Procedure

### Step 1
Open the Arduino official website:
https://www.arduino.cc/en/software

### Step 2
Download the Arduino IDE suitable for your operating system.

### Step 3
Run the installer.

### Step 4
Accept the license agreement.

### Step 5
Select all components and click Next.

### Step 6
Click Install and wait for installation to complete.

### Step 7
Launch Arduino IDE.

## Result
Arduino IDE was successfully installed and opened.

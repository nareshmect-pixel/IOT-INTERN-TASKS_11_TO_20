# Task 12 - Upload First Arduino Program

## Aim
To upload the first Arduino program and verify successful execution.

## Components Required
- Arduino Uno
- USB Cable
- Arduino IDE

## Theory
The Blink program is the simplest Arduino program. It turns the built-in LED ON and OFF repeatedly with a delay.

## Procedure

1. Open Arduino IDE.
2. Connect Arduino Uno to the computer.
3. Select the correct Board and Port.
4. Create a new sketch.
5. Enter the Blink code.
6. Click Verify.
7. Click Upload.
8. Observe the onboard LED blinking.

## Arduino Program
Refer to Blink_LED.ino

## Result
The built-in LED blinked continuously after successful upload.
